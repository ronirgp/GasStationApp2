Gas Station POS System

Overview

Gas Station POS System is a Python application built with SQLite for managing sales, inventory, employees, and business reports.

This project was created to practice Python programming, SQL databases, Git, and software development concepts.

Features

- User login system
- Password hashing for security
- Employee account management
- Sales tracking
- Cash register management
- Refund processing
- Receipt generation
- Inventory management
- Search inventory
- Update inventory quantities
- Restock inventory
- Delete inventory items
- Low stock report
- Out of stock report
- Sales revenue report
- Best selling product report
- Highest revenue product report
- Inventory summary report

Technologies Used

- Python
- SQLite
- Git
- GitHub

How to Run

1. Clone the repository.
2. Open the project folder.
3. Run:

python main.py
